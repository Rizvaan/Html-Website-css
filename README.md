# Html-Website-css

This is a responsive single-page website for a company that sells cylinders. The website is built with HTML and CSS.

## Project Description

The website has the following sections:
- **Home:** A home section with a call to action.
- **Solutions:** A section that displays the different solutions the company offers.
- **Products:** A section that displays the company's products.
- **Contact:** A section with the company's contact information.

The website is responsive and works on different screen sizes.

## Technologies Used

- HTML
- CSS

## How to use the project

To use the project, you can clone the repository and open the `index.html` file in your browser.
